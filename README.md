# Moonleaf Cafe Management System ☕🍰

A web-based cafe management system designed to manage menu items, orders, and customer data efficiently.

This project demonstrates backend development using **Python and Django**, along with frontend templates for managing cafe operations.

---

## Features

- Manage cafe menu items
- Add and update customer orders
- View order details
- Manage customer records
- Database integration for storing cafe data
- Backend logic using Django framework

---

## Technologies Used

- Python
- Django
- HTML / CSS
- SQLite Database
- Bootstrap (if used)

## How to Run the Project

### 1 Clone the repository


git clone https://github.com/Kritikagupt/MoonleafCafe.git


### 2 Navigate to project directory


cd MoonleafCafe


### 3 Install dependencies


pip install django


### 4 Run the server


python manage.py runserver


### 5 Open in browser


http://127.0.0.1:8000/
